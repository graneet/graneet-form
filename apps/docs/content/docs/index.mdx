# Welcome to Graneet Form 🚀

Graneet Form is a high-performance form library for React, designed to handle large-scale forms with ease. Built with TypeScript and performance in mind, it offers zero dependencies and a granular subscription system.

## Key Features

- **🚀 High Performance**: Handle forms with 1000+ fields without performance issues
- **🎯 Granular Subscriptions**: Only re-render components that actually need updates  
- **📦 Zero Dependencies**: Lightweight with no external dependencies
- **🗒️ Built-in Wizard System**: Native multi-step form support
- **🔧 TypeScript First**: Excellent developer experience with full type safety
